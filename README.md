Capy is a MERN web app that facilitates community by allowing users to create events and "pull up" to them!

# Technologies Used
MERN stack
MonogoDB
Express.js
React.js
Node.js

# Instructions to Run
```
add .env file to root
npm i 
npm run build
node server/server.js
```
